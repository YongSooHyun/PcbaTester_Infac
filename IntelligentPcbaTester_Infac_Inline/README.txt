파일 리스트

1. comport.cfg [선택]
COM PORT 설정을 보관한 파일.
만일 없으면 소스 코드 내에 보관된 디폴트 설정을 이용한다.

2. Crc32.NET.dll [필수]
CRC32 생성을 위해 소스코드에서 이용하는 라이브러리. 
현재는 grpMan.exe 를 이용하고 있으나, 추후 Data CRC 가 추가되면 필요하기 때문에 그대로 둠.

3. fn-dll.dll [필수]
Novaflash 디바이스와 통신하기 위한 라이브러리.

4. grpMan.exe [필수]
Novaflash GRP 파일의 GRP CRC32 를 계산하는데 이용되는 실행파일.

5. IntelligentPcbaTester.dll [필수]
이 프로그램을 eloZ1 TestBuilder에서 플러그인으로 실행시키기 위한 라이브러리.

6. IntelligentPcbaTester.dll.config [필수]
프로그램의 각종 설정 정보를 보관한 파일.

7. login.cfg [선택]
로그인 사용자 정보를 보관한 파일.
만일 없으면 관리자 권한을 가진 사용자이름 admin, 암호 admin 의 디폴트 유저가 생성된다.

8. products.cfg [선택]
Config -> Product Configuration 에서 저장한 제품 관련 설정(FGCODE, 프로젝트이름 등)이 
보관되는 파일.

9. eloZ1 함수 호출을 위한 라이브러리들. [필수]
- TestFramework.Common.dll
- TestFramework.PluginTestCell.dll

10. DMM 디바이스 연결을 위한 라이브러리들. [필수]
- Ivi.Visa.dll
- Keysight.Visa.dll

11. Script에서 이용하기 위한 Novaflash 연결 설정 및 GRP 파일 이름 보관 경로(Novaflash 테스트 스텝 스크립트에서 이용).
폴더 = C:\Users\[사용자이름]\AppData\Local\Temp\
파일 = ipts_nova_grp_files.temp

테스트 프로세스

1. 시작 버튼 누름
2. 설정 -> General에서 "테스트를 자동으로 다시 시작합니다." 체크박스가 설정되어 있고, 처음 시작이 아니라면 
START1 & START2 버튼 또는 STOP 버튼 누르라는 메시지 출력, 버튼 누름 대기(대기시간 0이면 바로 시작)
2.1. START1 & START2 버튼을 눌렀다면 다음 단계 시작, STOP 버튼 누르거나 타임아웃 되면 테스트 종료
3. 자동 모드인지 체크하고 아니라면 테스트 종료(설정에서 자동 모드 체크를 하지 않은 경우 이 단계 생략)
4. 픽스처 락 상태 체크, 락이 되어있지 않다면 "FIXTURE 가 Lock 되지 않았습니다. Lock 하겠습니까?" 메시지 출력
4.1. 사용자가 Yes 클릭하면 락 시도.
4.2. 락 실패이거나 사용자가 No 를 클릭했다면 테스트 종료
5. 탑 공압 센서 검사, 위에 있지 않다면 실린더 위로 올리기
6. 바코드 스캔 및 프로젝트 로딩
6.1. 바코드를 스캔해 해당 프로젝트를 로딩하도록 설정했다면 바코드 스캔 대기
6.1.1. 바코드를 파싱해 FGCODE 추출, FGCODE 에 대응하는 프로젝트 로딩
6.2. 바코드를 스캔하도록 설정하지 않았으면 프로젝트가 로딩됐는지 검사, 프로젝트가 로딩되지 않았으면 테스트 종료
7. DIO 디바이스에서 FID 읽어 프로젝트 FID 와 비교, 일치하지 않으면 테스트 종료
8. 안전센서 체크
9. Novaflash GRP CRC 계산, 프로젝트에 있는 GRP CRC와 비교하여 일치하지 않으면 테스트 종료
10. Novaflash GRP 파일이 Novaflash 디바이스에 있는지 검사, 없다면 업로드
11. Novaflash GRP 파일 리스트 및 Novaflash 연결 설정(주소, 시리얼인 경우 속도) 임시파일에 보관
12. 처음 시작이라면 START1 & START2 버튼 누름 대기(대기시간 0이면 바로 시작)
13. 실린더 다운
14. Probe Count 증가
15. 스크립트에서 DIO 액세스할 수 있도록 DIO 시리얼 닫기
16. eloZ1 실행
17. 중단 스위치가 눌렸으면 전원 차단, 실린더 상승, 버저음 출력 차례로 실행
18. 중단 스위치가 눌리지 않았으면 양품 불량품 루틴 수행
18.1. 양품, 불량품 상태 디스플레이
18.2. 양품, 불량품 상태에 따라 서로 다른 버저음 출력
18.3. 양품 수량 추가
18.4. 양품이면 실린더 상승
18.5. 불량이면 STOP 버튼 누르라는 메시지 출력하고 버튼 누르면 실린더 상승
18.6. 양품, 불량품 이력 저장
19. 모든 단계 종료, 2 로 이동
